# Sistema de Faculdade 🏫

Este projeto é um sistema de gerenciamento acadêmico desenvolvido em Java, com foco em funcionalidades como:

- Cadastro de alunos, professores e funcionários
- Gerenciamento de cursos (presenciais e online)
- Avaliação de desempenho (notas e provas)
- Geração de certificados
- Controle de matrículas

## 📚 Tecnologias Utilizadas

- Java
- IntelliJ IDEA
- Coleções (List, Map, Set)
- Programação Orientada a Objetos (POO)
- Tratamento de Exceções Personalizadas
